# Councils.gov

A static site built with Astro and USWDS (U.S. Web Design System).

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Technologies

- [Astro](https://astro.build/) - Static site generator
- [USWDS](https://designsystem.digital.gov/) - U.S. Web Design System
- [React](https://react.dev/) - UI library (available for components)
